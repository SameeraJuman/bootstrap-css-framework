# Kisskh Replica (UI Only)

This project is a **UI replica of the [kisskh](https://kisskh.co/)** website, built for front-end practice.

No backend logic, or real functionality is implemented.

### Demo link: https://sameerajuman.github.io/bootstrap-css-framework/kisskh-replica
---

## Purpose of the Project

- Practice responsive UI design with **Bootstrap**
- Strengthen HTML/CSS skills through real-world examples

---

## Languages used

- HTML5  
- CSS3  
- Bootstrap v5.3.8
