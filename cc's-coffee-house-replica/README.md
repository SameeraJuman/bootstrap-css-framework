# CC's Coffee House website replica
This project is a replica of the [CC's Coffee House](https://www.ccscoffee.com/) website, specifically the home page.

### Languages used
- HTML
- Bootstrap 5
- CSS 

### Why this project?
This project was built for my learning and better practice of my front-end development skills, more focusing on Bootstrap 5. 

### Features
- Responsive layout across desktop, tablet, and mobile screens
- Navigation bar 
- Hero section
- Adding text on top of a darkened image
- Bootstrap grid system and components
- Custom CSS
